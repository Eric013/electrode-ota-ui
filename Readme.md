wm-ota-ui
===
A CodePush Managementc UI.

Or run it 

```sh
  git clone 
  cd wm-ota-ui
  npm install
  npm run hot &
  open http://localhost:4000
```

##Installation
```sh
 $ npm install wm-ota-ui
``

##Running

Dev mode
```sh
 $npm run hot
 $ open http://localhost:4000
```


##Cli 
```sh
$ code-push login http://localhost.walmart.com:9001
```
